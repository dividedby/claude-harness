Prepare a structured handoff before manual compaction.

1. Update handoff.md with current state:
   - Summary (1-2 sentences)
   - What was done (files changed)
   - What is in progress
   - Decisions made and why
   - Exact resume instruction
2. Update TODO.md with current sprint status
3. Print "Ready to compact — run /compact now"

This preserves critical context that the compaction algorithm would otherwise discard.
