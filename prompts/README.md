# Prompts

Ready-to-paste prompts for common Claude Code scenarios.
Copy the contents of any file and paste directly into Claude.

| File | When to use |
|---|---|
| `migrate-harness.md` | Migrate an existing project to Harness v3 |
| `session-start.md` | Start of every Claude Code session |
| `new-feature.md` | Starting work on a new feature |
| `debug-issue.md` | Structured investigation of a bug or problem |
| `pr-prep.md` | Before opening a PR — review + done check |
| `quick-fix.md` | Small scoped fix without scope creep |
| `stack-decision.md` | Quick tech or tool decision |
| `end-of-session.md` | Capture state before ending a session |
